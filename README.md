# IDPA_22_Nr.3: Notenrechner
> Unser Projekt stellt einen Notenrechner für Schüler dar, worin man seine Noten einträgt und somit seinen Promotionsstand ermitteln kann. Basierend auf dem KBW Notenrechner.

## Technologie 
* Java 19.0.2+7-44
* Build Tool: Maven
* FXML
